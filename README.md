# Gate Pass - 通行证展示页面

一个简单的HTML页面，用于模拟展示校园或园区通行证。

## 项目特点

- 实时更新的日期和时间显示
- 个人信息展示区域（姓名、学工号、院系、专业、班级）
- 二维码展示区域
- 移动端友好的界面设计

## 文件结构

```
GatePass/
├── gatepass.html       # 主页面文件
├── HeadImgae.jpg       # 头像图片
├── ScreenShot.jpg      # 截图
├── screenshot.png      # 截图
└── DreenPass.gif       # 演示动图
```

## 使用方法

直接在浏览器中打开 `gatepass.html` 文件即可查看效果。

## 功能说明

1. **个人信息区域**：显示姓名、头像、学工号、院系、专业和班级信息
2. **扫码时间**：实时显示当前日期和时间，每秒更新一次
3. **二维码**：显示用于扫码验证的二维码图片
4. **底部时间**：大号字体显示当前时间

## 技术栈

- HTML5
- CSS3
- 原生JavaScript
